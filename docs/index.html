How Revenue is Attributed

// etl stages

Pipeline Flow

STAGE 01
Read & Validate
Schema-validated TSV read with Spark CSV reader. PERMISSIVE mode logs corrupt rows without failing the job.
spark.read.csv
STAGE 02
Parse Referrers
Spark UDFs extract search engine domain and keyword from the referrer URL on every hit row.
@udf(StringType)
STAGE 03
Session Attribution
Window function propagates first-touch search keyword forward across all hits in a session.
Window.partitionBy
STAGE 04
Filter Purchases
Regex match for standalone event "1" — avoids false matches against events 10, 11, 12, 13, 14.
rlike regex
STAGE 05
Parse Revenue
Explodes semicolon-delimited product_list, extracts 4th field (revenue), sums per hit.
F.explode
STAGE 06
Aggregate & Write
Groups by engine + keyword, sums revenue, sorts descending. Writes tab-delimited output to S3.
.groupBy().agg()
